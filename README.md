 # 🧮 CalCraft

A modern, responsive calculator web application built using **HTML5**, **CSS3**, and **JavaScript**. CalCraft provides a clean and user-friendly interface for performing basic arithmetic operations with keyboard support and responsive design.

---

## 🌐 Live Demo

**Vercel Deployment:**  
https://cal-craftt.vercel.app/

---

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📊 Modulus (%)
- 🔢 Decimal Calculations
- 🧹 Clear (AC)
- ⌫ Delete (DEL)
- ⌨️ Keyboard Support
- 📱 Responsive Design
- 🎨 Modern User Interface
- ⚡ Fast and Lightweight

---

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Mdsahil01/cal-craft.git
```

### Open the Project

```bash
cd cal-craft
```

Open `index.html` in your preferred web browser.

## 💻 Calculator Operations

| Operation | Symbol |
|-----------|--------|
| Addition | + |
| Subtraction | - |
| Multiplication | × |
| Division | ÷ |
| Modulus | % |
| Decimal | . |
| Clear | AC |
| Delete | DEL |
| Calculate | = |

---

## ⌨️ Keyboard Shortcuts

| Key | Action |
|------|--------|
| 0–9 | Enter numbers |
| + - * / % | Arithmetic operators |
| . | Decimal point |
| Enter | Calculate result |
| Backspace | Delete last character |
| Escape | Clear calculator |
